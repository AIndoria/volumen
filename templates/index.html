a personalized instance of termpad, self-hosted pastebin, customized for Civitates Res Publica.

Usage

Upload some text:

From the web

With cURL: 

            $ curl -d "Germania Delenda Est\!" {{domain}}
            $ curl --data-binary @path/to/file {{domain}}
fish function(place this as volumen.fish into your ~/.config/fish/functions folder): 

            function volumen
                curl --data-binary @/dev/stdin https://volumen.civitat.es
            end
            $ cat reasonsGermansSuck.file | volumen
With HTTPie: 

            $ echo "Germania Delenda Est\!" | http POST {{domain}}
            $ http POST {{domain}} < path/to/file

Get some text:

With cURL: 

            $ curl {{domain}}/raw/PlainVastAirport
With HTTPie: 

            $ http {{domain}}/raw/IdioticGermanCivilians
Don't forget to add that /raw/ there or else you will get html code meant for the browser
